# IPL Performance Analysis – RCB

This project analyzes the performance of Royal Challengers Bangalore (RCB) in the Indian Premier League (IPL) using match-level and ball-by-ball datasets from 2008–2022.

## Objectives
- Evaluate overall team performance
- Analyze impact of toss decisions
- Compare batting strategies (batting first vs chasing)
- Identify challenging opponents
- Assess dependency on key players

## Tools & Technologies
- Python (Pandas, Matplotlib)
- Jupyter Notebook

## Key Insights
- RCB win rate: 48.23%
- Performs better when batting first than chasing
- Toss has only a marginal impact on outcomes
- Struggles against strong teams like CSK and MI
- Win probability increases when Virat Kohli scores 50+

## Project Structure
- `RCB_Analysis.ipynb` – Main analysis notebook

## Conclusion
RCB shows inconsistent performance across seasons, with stronger results when defending totals. Performance gaps against top teams and reliance on key players highlight areas for improvement.
